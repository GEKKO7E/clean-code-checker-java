## 📖 About The Project

Clean Code Java Checker is a web application that assists developers and students in reviewing Java source code more efficiently.

The system provides automated code evaluation and generates detailed reports covering code quality, security concerns, performance observations, and improvement suggestions. By presenting analysis results in a structured dashboard, users can quickly identify potential issues and understand areas that require refactoring or optimization.

The application is designed to create a simple, modern, and accessible environment for code review activities without requiring complex software installation.
